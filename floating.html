position:relative; possible to specify distance from sides: top:20px; moves it down 20px.
left:50px moves it to the right 50px. position:absolute; ignores the rest of the flow and overlaps.

position:fixed; the div stays in the same place on the screen when you scroll up and down. This is used for menues.

The default position is static.

margin:50px; creates a margin of 50px around the div, so all other elements move accordingly.
if we assign 4 measurments, the first is the top, and the others follow clockwise.
possible to assign margin to selected sides: margin-right:, margin-top:xx;

padding is the space around the text. we to set the p tag to zero and margin to 0 in order to be able
to control the padding fully.
If we write one value after the padding:, it will give that measure on four sides of the text.
If we write 2 values, they will repeat themselve, 1st top, 2nd right, 1st bottom, 2nd bottom.
We can also use padding-top:x; etc.
The measurments of the padding are added to the width and height we defined for the div.
so if we gave 10px padding, and the width is 200px, the final div will be 220px.
